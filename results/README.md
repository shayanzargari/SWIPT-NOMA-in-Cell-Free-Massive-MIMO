# Results

This directory is used for generated numerical outputs.

Run:

```bash
python scripts/rebuild_fig2.py --mc 500
```

Generated file:

```text
figure2_results.csv
```

The CSV contains:

- `users`: number of users
- `swipt_noma`: ergodic sum rate for SWIPT-NOMA
- `noma`: ergodic sum rate for conventional NOMA
- `oma`: ergodic sum rate for OMA
